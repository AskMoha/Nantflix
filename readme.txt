Bonjour bienvenue sur Nantflix
pour se connecter voici ce qu'il vous faut :
L'identifiant est "root"
il n'y a pas de mot de passe
le nom de la base de donnée est "nantflix"
le serveur est "localhost"
le port utilisé est le 3306
Le fichier de création de base de données est nantflix.sql, il existe déjà une série et deux utilisateur pour montrer que tout fonctionne bien 
Les vidéos des differentes séries sont stockées dans les dossiers 1 2 etc avec le numéro correspondant à l'id de la série 
La page de base est base.html le lien pour y acceder est : http://localhost/projet/base.html
